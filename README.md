Nombre del proyecto: Portafolio de proyectos Francisco Guadarrama

Descripción del proyecto: Se construyó un portafolio de proyectos con la finalidad de dar a conocer las diferentes paginas web creadas, así como las técnicas y tecnologías empleadas. En este portafolio también se implementaron accesos de interacción o de contacto con la finalidad de poder establecer comunicación con cualquier persona que este interesada ya sea en conocer mi trabajo o en entablar comunicación para trabajar en equipo. En este caso cuenta solo con dos proyectos pero se irán anexando más en un futuro.

Las tecnologías que se implementaron en el proyecto son muy variadas, para la creación del portafolio se utilizo HTML5 y CSS3, también se utilizo la implementación de la IA para abordar los temas de aprendizaje de diversas técnicas como el uso de los elementos ::after o ::before, la implementación de efectos como sombreados, degradados y efectos de luz. Así como el uso de media queries para realizar un dinamismo en la página asegurando la responsividad de esta para diferentes dispositivos.
